## üéØ The 4 New Endpoints (November 2025)

These are the most important endpoints for MapTool ‚Üí Adamo integration:

1. `POST /adamo/initial` - Create MAP_INITIAL
2. `POST /adamo/session` - Create MAP_SESSION
3. `POST /adamo/result` - Create MAP_RESULT
4. `POST /adamo/session-with-results` ‚≠ê - **RECOMMENDED** (atomic transaction)

**See:** [endpoints/MAP_RESULT_ENDPOINTS.md](./endpoints/MAP_RESULT_ENDPOINTS.md) for details

## üí° Key Concept

All 9 MapTool evaluation types map to the same Adamo endpoint - just change `stage`, `subStage`, and `segment` parameters!

**See:** [guides/MAPTOOL_TO_ADAMO_INTEGRATION_GUIDE.md](./guides/MAPTOOL_TO_ADAMO_INTEGRATION_GUIDE.md) for mapping details
